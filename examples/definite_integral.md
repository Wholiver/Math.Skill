# 定积分：$\displaystyle \int_0^{\pi/2} \cos^2 x \, dx$ 的计算

## 用户输入
计算定积分 $\displaystyle \int_0^{\pi/2} \cos^2 x \, dx$

## Skill 分类
微积分 / 定积分

## 题意解析
- **被积函数**：$\cos^2 x$，是三角函数的平方
- **积分区间**：$[0, \pi/2]$，余弦函数在该区间上非负
- **目标**：计算精确的定积分值
- **方法选择依据**：$\cos^2 x$ 需要通过三角恒等式降次
- **解的约束**：定积分的值应为正实数（被积函数非负），且不超过区间长度 $\pi/2$（$\cos^2 x \leq 1$）

## 方法选择

**主方法：三角恒等式降次 + 直接积分**

使用倍角公式 $\cos^2 x = \dfrac{1 + \cos 2x}{2}$，将二次三角函数转化为一次三角函数，然后直接积分。

选择理由：对于 $\cos^2 x$ 或 $\sin^2 x$ 的形式，降次是最标准的处理方式，计算简单且不容出错。

**备选方法**：
1. **Wallis 公式**：$\int_0^{\pi/2} \cos^n x \, dx$ 的递推公式
2. **$\beta$ 函数**：化为 $B(1/2, 3/2)/2$ 计算
3. **对称性**：$\int_0^{\pi/2} \cos^2 x \, dx = \int_0^{\pi/2} \sin^2 x \, dx$，且二者之和为 $\pi/2$，故各为 $\pi/4$

本文展示主方法和对称性法，并用中点法做数值验证。

## 解题过程

### 方法一：三角恒等式降次

**Step 1：应用倍角公式**

$$\cos 2x = 2\cos^2 x - 1 \implies \cos^2 x = \frac{1 + \cos 2x}{2}$$

验证恒等式：$2\cos^2 x - 1 = \cos^2 x - \sin^2 x = \cos 2x$ ✓

**Step 2：代入积分**

$$\int_0^{\pi/2} \cos^2 x \, dx = \int_0^{\pi/2} \frac{1 + \cos 2x}{2} \, dx$$

$$= \frac{1}{2} \int_0^{\pi/2} 1 \, dx + \frac{1}{2} \int_0^{\pi/2} \cos 2x \, dx$$

**Step 3：分别计算**

第一项：
$$\frac{1}{2} \int_0^{\pi/2} 1 \, dx = \frac{1}{2} \cdot \left[x\right]_0^{\pi/2} = \frac{1}{2} \cdot \frac{\pi}{2} = \frac{\pi}{4}$$

第二项：
$$\frac{1}{2} \int_0^{\pi/2} \cos 2x \, dx = \frac{1}{2} \cdot \left[\frac{\sin 2x}{2}\right]_0^{\pi/2} = \frac{1}{4} \left[\sin 2x\right]_0^{\pi/2}$$
$$= \frac{1}{4} (\sin \pi - \sin 0) = \frac{1}{4} (0 - 0) = 0$$

**Step 4：汇总**

$$\int_0^{\pi/2} \cos^2 x \, dx = \frac{\pi}{4} + 0 = \frac{\pi}{4}$$

### 方法二：对称性法（备选验证）

**Step 1：利用对称性**

在 $[0, \pi/2]$ 上，令 $u = \pi/2 - x$：

$$\int_0^{\pi/2} \cos^2 x \, dx = \int_{\pi/2}^0 \cos^2(\pi/2 - u) \cdot (-du)$$
$$= \int_0^{\pi/2} \sin^2 u \, du$$

所以 $\int_0^{\pi/2} \cos^2 x \, dx = \int_0^{\pi/2} \sin^2 x \, dx$

**Step 2：求和**

$$\int_0^{\pi/2} \cos^2 x \, dx + \int_0^{\pi/2} \sin^2 x \, dx = \int_0^{\pi/2} (\cos^2 x + \sin^2 x) \, dx$$
$$= \int_0^{\pi/2} 1 \, dx = \frac{\pi}{2}$$

由于两个积分相等，各为 $\pi/4$。即 $\int_0^{\pi/2} \cos^2 x \, dx = \frac{\pi}{2} \div 2 = \frac{\pi}{4}$ ✓

## 验算

**验算方法 1：中点法数值近似（$n = 10$）**

将 $[0, \pi/2]$ 等分为 $n = 10$ 个子区间，每个宽度 $\Delta x = \pi/20 \approx 0.15708$。

中点法公式：$\int_a^b f(x) \, dx \approx \Delta x \sum_{i=1}^n f\left(\frac{x_{i-1} + x_i}{2}\right)$

近似值 $= 0.15708 \times 4.93357 \approx 0.7750$

精确值 $\pi/4 \approx 0.785398$，相对误差 $\approx 1.3\%$（$n=10$ 时中点法的预期精度）✓

**验算方法 2：Wallis 公式验证**

$$\int_0^{\pi/2} \cos^{2} x \, dx = \frac{2-1}{2} \cdot \frac{\pi}{2} = \frac{1}{2} \cdot \frac{\pi}{2} = \frac{\pi}{4}$$

与结果一致 ✓

**验算方法 3：区间合理性检查**

被积函数 $\cos^2 x$ 在 $[0, \pi/2]$ 上满足 $0 \leq \cos^2 x \leq 1$，因此：
$$0 \leq \int_0^{\pi/2} \cos^2 x \, dx \leq \frac{\pi}{2}$$

$\pi/4 \approx 0.7854$ 在 $[0, 1.5708]$ 内 ✓

此外，$\cos^2 x$ 在 $[0, \pi/2]$ 上的平均值为 $\frac{\pi/4}{\pi/2} = \frac{1}{2}$，这与 $\cos^2 x + \sin^2 x = 1$ 且两函数图像对称的直观一致 ✓

## 最终答案
$$\int_0^{\pi/2} \cos^2 x \, dx = \frac{\pi}{4}$$

## 易错点
1. **定积分上下限代入时漏算**：计算 $\frac{1}{4}[\sin 2x]_0^{\pi/2}$ 时注意 $\sin \pi = 0$（不是 $1$），$\sin 0 = 0$
2. **倍角公式符号**：$\cos 2x = 2\cos^2 x - 1 = 1 - 2\sin^2 x$，注意变体不要混淆
3. **降次后的系数**：$\cos^2 x = \frac{1 + \cos 2x}{2}$，积分后还要乘以 $\frac{1}{2}$ 的因子，容易漏掉
4. **对称性法的前提**：$\int_0^{\pi/2} \cos^2 x \, dx = \int_0^{\pi/2} \sin^2 x \, dx$ 仅在 $[0, \pi/2]$ 上成立（变量代换 $u = \pi/2 - x$），不能随意推广到其他区间
5. **数值验证的意义**：$n=10$ 的中点法误差约 1%，用于粗略验证而非精确证明；要确信结果正确应以解析推导为准
