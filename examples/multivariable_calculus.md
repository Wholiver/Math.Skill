# 多元函数临界点与分类

## 用户输入
求函数 $f(x, y) = x^3 + y^3 - 3xy$ 的所有临界点，并判断每个临界点的类型（极大值点、极小值点或鞍点）。

## Skill 分类
多元微积分

## 题意解析
- **函数形式**：$f(x, y) = x^3 + y^3 - 3xy$，定义域为 $\mathbb{R}^2$。
- **求解目标**：(1) 所有临界点（即 $\nabla f = \mathbf{0}$ 的点）；(2) 分类各临界点。
- **关键定理**：二阶导数检验（Hessian 行列式判别法）。对临界点 $(x_0, y_0)$，计算

  $$D = f_{xx}(x_0, y_0) \cdot f_{yy}(x_0, y_0) - [f_{xy}(x_0, y_0)]^2$$

  - 若 $D > 0$ 且 $f_{xx} > 0$：极小值点
  - 若 $D > 0$ 且 $f_{xx} < 0$：极大值点
  - 若 $D < 0$：鞍点
  - 若 $D = 0$：无法判定

- **隐含条件**：$f$ 是 $C^\infty$ 函数（无限阶可导），所有偏导数存在且连续，二阶导数检验条件满足。

## 方法选择
**首选方法**：梯度为零求临界点 + Hessian 矩阵行列式分类。这是多元函数极值问题的标准流程。

**备选方法**：Taylor 展开到二阶项判断二次型的正定性。本质与 Hessian 判别等价。

## 解题过程

### 第一步：求一阶偏导数并令其为零

$$f_x = \frac{\partial f}{\partial x} = 3x^2 - 3y$$

$$f_y = \frac{\partial f}{\partial y} = 3y^2 - 3x$$

令 $\nabla f = \mathbf{0}$：

$$\begin{cases}
3x^2 - 3y = 0 \\
3y^2 - 3x = 0
\end{cases}
\implies
\begin{cases}
x^2 = y \\
y^2 = x
\end{cases}$$

### 第二步：求解方程组

将 $y = x^2$ 代入 $y^2 = x$：

$$(x^2)^2 = x \quad \Rightarrow \quad x^4 = x \quad \Rightarrow \quad x^4 - x = 0 \quad \Rightarrow \quad x(x^3 - 1) = 0$$

$$x(x - 1)(x^2 + x + 1) = 0$$

实数解：$x = 0$ 或 $x = 1$（$x^2 + x + 1 = 0$ 无实根，判别式 $1 - 4 = -3 < 0$）。

- 当 $x = 0$：$y = x^2 = 0$，临界点为 $(0, 0)$。
- 当 $x = 1$：$y = x^2 = 1$，临界点为 $(1, 1)$。

### 第三步：求二阶偏导数

$$f_{xx} = \frac{\partial^2 f}{\partial x^2} = 6x$$

$$f_{yy} = \frac{\partial^2 f}{\partial y^2} = 6y$$

$$f_{xy} = f_{yx} = \frac{\partial^2 f}{\partial x \partial y} = -3$$

### 第四步：对每个临界点计算 Hessian 行列式

**临界点 $(0, 0)$**：

$$f_{xx}(0,0) = 0, \quad f_{yy}(0,0) = 0, \quad f_{xy}(0,0) = -3$$

$$D(0,0) = 0 \cdot 0 - (-3)^2 = 0 - 9 = -9 < 0$$

$D < 0$，所以 $(0, 0)$ 是**鞍点**。

**临界点 $(1, 1)$**：

$$f_{xx}(1,1) = 6, \quad f_{yy}(1,1) = 6, \quad f_{xy}(1,1) = -3$$

$$D(1,1) = 6 \cdot 6 - (-3)^2 = 36 - 9 = 27 > 0$$

$D > 0$ 且 $f_{xx}(1,1) = 6 > 0$，所以 $(1, 1)$ 是**极小值点**。

极小值为：

$$f(1, 1) = 1^3 + 1^3 - 3 \cdot 1 \cdot 1 = 1 + 1 - 3 = -1$$

## 验算

### 方法一：验证梯度在临界点处为零

$(0, 0)$ 点：$f_x(0,0) = 3(0)^2 - 3(0) = 0$，$f_y(0,0) = 3(0)^2 - 3(0) = 0$。✓

$(1, 1)$ 点：$f_x(1,1) = 3(1)^2 - 3(1) = 0$，$f_y(1,1) = 3(1)^2 - 3(1) = 0$。✓

### 方法二：函数值比较法

在 $(0, 0)$ 附近取测试点：

- 沿 $y = x$ 方向：$f(t, t) = 2t^3 - 3t^2$。当 $t$ 很小时，$f(t, t) \approx -3t^2 < 0 = f(0,0)$（$t \neq 0$）。
- 沿 $y = -x$ 方向：$f(t, -t) = t^3 + (-t)^3 - 3t(-t) = 0 + 3t^2 = 3t^2 > 0 = f(0,0)$（$t \neq 0$）。

函数值在 $(0,0)$ 附近既有大于 $f(0,0)$ 的也有小于 $f(0,0)$ 的，确认为鞍点。✓

在 $(1, 1)$ 附近取测试点，二阶 Taylor 展开：

$$f(1+h,1+k) \approx f(1,1) + \frac{1}{2}(6h^2 + 6k^2 - 6hk) = -1 + \frac{3}{2}((h-k)^2 + h^2 + k^2) > -1$$

对于 $(h,k) \neq (0,0)$，函数值大于 $-1$，确认为极小值点。✓

## 最终答案

$$\boxed{\text{临界点：}(0, 0) \text{ 和 } (1, 1)}$$

$$\boxed{(0, 0) \text{ 是鞍点}}$$

$$\boxed{(1, 1) \text{ 是极小值点，极小值 } f_{\min} = -1}$$

## 易错点
1. **遗漏 $x = 0$ 解**：解 $x^4 = x$ 时不要把 $x = 0$ 约掉（即不能两边同时除以 $x$），应移项写成 $x^4 - x = 0$，因式分解求得全部解。
2. **Hessian 行列式的符号判断**：$D > 0$ 且 $f_{xx} > 0$ 是极小值；$D > 0$ 且 $f_{xx} < 0$ 是极大值。有些教材使用 $-D$ 的判别，注意正负号方向。
3. **仅有两个临界点**：别被方程组 $x^2 = y$ 和 $y^2 = x$ 的对称性误导以为有无穷多解。通过代入消元可知只有两个实数解。
4. **$D = 0$ 的情况**：若 Hessian 行列式为零，二阶导数检验失效，需使用更高阶展开或其他方法判定。本题不涉及此情况。
